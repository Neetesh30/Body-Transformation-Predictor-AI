Body-Transformation-Predictor-AI

Description
This project is a web-based BMI calculator enhanced with AI capabilities that analyzes a user's body metrics and uploaded image to generate a visual transformation preview based on a selected diet and fitness plan.

It helps users understand how their body may change over time if they follow a structured health program.

Features
- BMI (Body Mass Index) calculation
- User profile and image upload
- AI-based body transformation visualization
- Diet plan selection (weight loss / muscle gain / maintenance)
- Before & after comparison view
- Send export data in pdf format data to user using whatsapp API 
- Responsive UI for mobile and desktop

Tech Stack
PHP / Laravel
MySQL
HTML, CSS, JavaScript
Bootstrap
AI Image Processing (OPENAI,W HATSAPP API)
